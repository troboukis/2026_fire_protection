ROOT CAUSE ANALYSIS
================================================================================

CRITICAL ISSUE 1: Fire Incidents Municipality Name Encoding
Problem:      Fire municipality names use uppercase without diacritics
              (e.g., "ΑΘΗΝΑΣ") while mapping uses proper Greek formatting
              (e.g., "Αθηναίων")
Affected:     720 municipality entries (100% of fire_incidents municipalities)
Impact:       Fire incident data cannot be linked to municipality coverage
Solution:     Normalize fire municipality names through accent/case conversion
              OR create a mapping table for fire municipality name variants

CRITICAL ISSUE 2: Diavgeia Organization JSON Parsing
Problem:      The 'organization' column contains JSON-serialized dict objects
              that are not being parsed correctly. Some entries (406) appear
              to be malformed or unparseable JSON strings.
Affected:     406-341 entries in diavgeia
              63 organizations missing from mapping (from parsed entries)
Impact:       Incomplete extraction of organization data from diavgeia
Solution:     Use proper JSON parsing (json.loads) for organization field
              Add error handling for malformed JSON
              Verify why org_name_clean has all orgs but organization field
              extraction fails for some

MINOR ISSUE 3: Missing Organization in Raw Review File
Problem:      "ΔΕΥΑ ΔΉΜΟΥ ΤΕΜΠΩΝ" exists in raw_unmapped_orgs review but
              not in final_entity_mapping
Affected:     1 organization
Impact:       One reviewed org was not added to final mapping
Solution:     Add this org to final_entity_mapping (appears to map to
              municipality_id 9101 in Thessaly region)

MINOR ISSUE 4: Geojson Municipality Code Mismatch
Problem:      Mapping coverage includes municipality_id 9305 but this ID
              doesn't exist in municipalities.geojson
Affected:     1 municipality ID
Impact:       Geographic data missing for one municipality ID
Solution:     Verify if 9305 should be in geojson or removed from mapping
